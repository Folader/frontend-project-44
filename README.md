## Описание

Набор из 5 консольных математических игр для тренировки мозга. Каждая игра задает 3 вопроса, для победы нужно правильно ответить на все.

## Минимальные требования

- Node.js версии 13 и выше
- npm

## Установка

```bash
git clone https://github.com/Folader/frontend-project-44.git
cd frontend-project-44
npm install
npm link
```

## Доступные игры

### Brain Even
Определите, является ли число четным.

```bash
brain-even
```
